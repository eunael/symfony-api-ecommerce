# Symfony (API Platform): API E-commerce

## Configuração

### 1. Clone o projeto

### 2. Configure o Docker

```sh
docker compose build --no-cache
docker compose up --wait
```

### 3. Configure os Git Hooks

```sh
git config core.hooksPath .githooks
```

### 4. Crie o `.env.local`

```sh
cp api/.env api/.env.local
```
